# Dongwon-KAIST-AI-Competition

## 프로젝트 개요

2025 동원 x 카이스트 AI Competition: Unlocking Future Sales & Demographics

제품별 판매량을 예측하기 위해 페르소나 기반 에이전트 기반 모델링(ABM)과 배스 확산 모델을 사용하는 프로젝트입니다.

## 주요 기능

- Gemini API를 활용한 페르소나 생성
- 페르소나 캐시 시스템 (JSON 파일 저장/로드)
- ABM (Agent-Based Modeling) 시뮬레이션
- 배스 확산 모델 적용
- 신제품 출시일 기반 판매량 조정
